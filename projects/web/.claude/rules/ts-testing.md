# TypeScript/JavaScript Validation

> This file extends [testing.md](./testing.md) with TypeScript/JavaScript specific content.

## Browser-Based Validation

Use **Playwright MCP** for functional validation of web apps — real browser, real UI interactions.

## Validation Approach

1. Build and start the dev server
2. Navigate to the feature in a real browser
3. Exercise all user flows through actual UI interaction
4. Capture screenshots and network responses as evidence
5. Verify behavior matches expectations
